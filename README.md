# 明日方舟自动刷图脚本 v0.01

基于Python 3.7.3 64-bit

请在windows上运行模拟器，强烈建议设置模拟器窗口大小为1440x960,同时请确保模拟器不会卡顿，因为脚本是根据时间来运行的。
在编译器中运行代码时，请以管理员权限运行；运行独立exe时则不需要这样做。（pyinstaller打包出来的exe是真的大啊）


现在具有的功能:
  1. 自定义刷图次数
  2. 自定义每张图的时间
  3. 支持后台刷图（工作原理是每次过完图切出模拟器，点击完成后再切会之前的应用）
